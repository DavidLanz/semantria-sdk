Semantria C++ SDK offers the most easiest way to start using of Semantria text analytic service on C/C++.
SDK is written on native C++ and consist of API utilization code, unit tests and simple demo app.

http://semantria.com/documentation

===================================================================================================
Use Code::Blocks application(http://www.codeblocks.org/) to build the project.
It's free, cross-platform IDE.

Code::Blocks does not, by default, use Makefiles it has its own .cbp files which do the same thing. 
More information about Makefiles in Code::Blocks here:
http://wiki.codeblocks.org/index.php?title=Code::Blocks_and_Makefiles
